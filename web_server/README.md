# Web Server Project
This directory contains scripts to automate the configuration of an Nginx web server on Ubuntu 16.04.
- `0-transfer_file`: Script to transfer files via SCP.
- `1-install_nginx_web_server`: Installs Nginx and sets a basic welcome message.
- `2-setup_a_domain_name`: Contains the domain name.
- `3-redirection`: Configures a 301 permanent redirect.
- `4-not_found_page_404`: Configures a custom 404 error page.
